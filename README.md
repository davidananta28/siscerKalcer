# Diagnosis Penyakit Kanker Paru Paru Berbasis Fuzzy Inference System

## Prasyarat (Requirements)
Sebelum menjalankan aplikasi, pastikan **Python** telah terinstall pada perangkat. Library yang dibutuhkan adalah:

- **Flask** – Framework web
- **NumPy** – Pengolahan data numerik
- **Scikit-Fuzzy** – Pemrosesan logika fuzzy

Instal semua library sekaligus dengan perintah berikut:

```bash
pip install flask numpy scikit-fuzzy
```

---

## Cara Menjalankan Aplikasi

### Siapkan Struktur Folder

Pastikan struktur file proyek terlihat seperti berikut:

```text
project-root/
│
├── app.py              # Kode utama aplikasi Flask
├── templates/
│   └── index.html      # File HTML antarmuka web
└── README.md
```

### Jalankan Server

Buka terminal atau command prompt pada direktori proyek, lalu jalankan:

```bash
python app.py
```

### Akses Aplikasi

Buka browser dan akses alamat berikut:

```text
http://127.0.0.1:5000
```
